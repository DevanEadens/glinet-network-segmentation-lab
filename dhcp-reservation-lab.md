# GL.iNet Network Segmentation Lab (GL-MT300N-V2)

## Overview
Configured a GL.iNet GL-MT300N-V2 travel router to create a segmented wireless network with isolated guest access.

---

## What I Built
- Main Wi-Fi network for trusted devices
- Guest Wi-Fi network for isolated access
- Secure wireless configuration using WPA2/WPA3

---

## Key Skills Demonstrated
- Network segmentation
- Wireless security configuration
- Router administration
- Client isolation concepts

---

## Validation
- Main network: Full admin access to router
- Guest network: Internet access only, no admin access
